Both Entities.cs and OptionSets.cs are auto generated by CRM SDK.

To regenerate:
	- go to /Sample.CrmSdkTools/GenerateCrmEntities/
	- open GenerateEntitiesAndOptionSets.bat and ensure that the file is using the correct connection and namespace settings.
	- run GenerateEntitiesAndOptionSets.bat by double clicking.
	- copy the files from the /Sample.CrmSdkTools/GenerateCrmEntities/Generated/ folder into this folder.
